
# 📊 Dashboard de Vendas no Excel

## 📌 Sobre o Projeto
Este projeto apresenta um dashboard de vendas desenvolvido no Excel com o objetivo de transformar dados brutos em informações visuais claras e úteis para análise de desempenho.

O dashboard permite acompanhar métricas importantes de vendas e identificar tendências que auxiliam na tomada de decisões baseada em dados.

---

## 🎯 Objetivo
Criar um dashboard que permita:

- Visualizar o desempenho geral de vendas
- Analisar vendas por produto
- Identificar tendências mensais
- Facilitar a interpretação de dados através de gráficos

## 📊 Funcionalidades

O dashboard inclui:

- Indicadores de vendas totais
- Análise de vendas por produto
- Tendência mensal de vendas
- Visualização por gráficos

## 🛠️ Tecnologias Utilizadas

- Microsoft Excel
- Gráficos
- Fórmulas de análise de dados
